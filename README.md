# World Wide Energy Consumption Analysis using SQL

## Overview
This project analyzes global energy consumption, production, CO₂ emissions, GDP, and population using MySQL. The goal is to identify trends, compare countries, and generate meaningful insights through SQL queries.

## Technologies
- MySQL
- SQL
- MySQL Workbench

## Dataset
- Country
- Consumption
- Production
- Emission
- GDP
- Population

## Key Analysis
- Top emitting countries
- Top GDP countries
- Energy production vs consumption
- Global emission trends
- Emission-to-GDP ratio
- Global emission share
